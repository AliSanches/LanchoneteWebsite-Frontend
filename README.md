# Lanchonete

## Bibliotecas Utilizadas

- Bootstrap v5.3.3
- React Bootstrap v2.10.5
- React Router Dom
- React Icons

## Requisitos

- Node versão 22

## Instalação

- Clonar o repositório para sua máquina local
- Abrir o projeto no Vscode
- Instalar as dependências
- Digitar "npm run dev" e abrir no navegador
